# **App Name**: URA Web Canvas

## Core Features:

- HTML Canvas Creation: Enable users to create HTML code visually through a drag-and-drop interface, adding various design elements to the canvas.
- Start from Templates: Offer users a choice between starting with a blank page or pre-designed document templates to accelerate the design process.
- Interactive Design Tools: Provide a rich set of tools including adding and customizing buttons, text, images, and animations with options to resize, recolor, and apply effects.
- Multi-Page Management: Allow users to create and manage multiple pages within a project, with easy navigation and the ability to set page-specific colors and redirection settings.
- Animation and Effect AI Tool: A tool that will use reasoning to pick relevant, page-appropriate, professional looking animations and effects to suggest for elements on the page.
- Project Saving and Loading: Implement a save project feature where users can save their projects to Firestore using a unique 6-digit ID, allowing them to load and continue editing their designs later.
- Image Importing with URL: Allow users to import images into their projects by providing a URL.

## Style Guidelines:

- Primary color: Soft blue (#A0CFEC) for a calm, creative feel.
- Background color: Light gray (#F0F4F7) for a clean, modern canvas.
- Accent color: Muted orange (#E59866) for interactive elements and highlights.
- Font: 'Inter' sans-serif for a clean and modern design across the application.
- Use a set of consistent, minimalist icons for design tools and page navigation.
- A clean and intuitive layout that enables easy access to all features and tools.
- Use subtle and smooth animations to enhance user experience when adding elements and navigating pages.
